# Khởi chạy Frontend

Giao diện người dùng được xây dựng bằng ReactJS, đây là giao diện người dùng sẽ tương tác chủ yếu

## Hướng dẫn cài đặt

1. **Cài đặt thư viện:**

   ```bash
   npm install
   ```

2. **Khởi chạy:**

   ```bash
   npm run dev
   ```

3. **Truy cập:**
   Ứng dụng sẽ chạy tại: `http://localhost:5173`
